# Change Log
All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning](http://semver.org/).


## [0.4.0] - 2016-07-07
### Added
- Added a new metric to control the real memory used (without buffer and cache)
- Added a new graph on the GUI to visualize this new metric

## [0.3.0] - 2016-05-02
### Added
- New Threshold metric IP allows to see how many ips are connected to the server
- Added the script that is running on each server to get the server data
- Added color badge for the IP metric on the mkit app gui

## [0.2.0] - 2016-04-26
### Added
- Threshold table to keep information of the server thresholds
- Cron job to send email notifications on server thresholds
- Added color badges to make more visual the connections status on the servers


## [0.1.0] - 2015-12-06
### Added
- First Mkit Server Status App
- Allow to view disk, memory and cpu use
